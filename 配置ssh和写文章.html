配置SSH和hexo发布文章

发表于 2017-09-25

##配置SSH
在git中执行下面命令生成SSH

ssh-keygen

  • 敲3次回车 生成id_rsa.pub,记事本打开复制SSH key的内容
  • 打开github，点击setting-SSH keys— Add SSH
  • 把复制的内容粘贴到下面框中，title随便填写

判断SSH是否配置好

ssh -T git@github.com

若出现
Hi XX! You've successfully authenticated,but Github does not provide shell access.
代表配置完成

发布文章

hexo new “title”

在本地hexo->source->_post目录下可以看到你新建的文章

之后可以利用markdown编辑器编辑文章 http://www.appinn.com/markdown/

在文章中插入图片和修改图片，推荐使用图库，域名[sm.ms]

写好了 在git中依次执行hexo clean hexo generate hexo deploy就可以发布了
